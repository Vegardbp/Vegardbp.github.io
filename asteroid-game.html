Roguelike Asteroid Game

C++ · threepp · VCPkg · Git

A 2.5D rougelike take on Atari Asteroids, built in (almost) pure C++ from scratch.

Built as a final project for the Object Oriented Programming course at NTNU. Graded A with a focus on code quality, architecture, and design patterns.

Features a custom-built physics engine, particle system, roguelike item modifiers, and more.

All code contributions, and 3D art are by me, but uses some AI-generated assets, which I now regret using.

Source code on GitHub

Details

Custom physics engine 3D collision detection and response built from scratch in C++, with ability to inject collision logic via callbacks
Particle system Custom particle system for explosions, thrust, visual effects. Also used to spawn all asteroids, using inheritance from particle base class
Roguelike elements Asteroids drop random items when destroyed, which apply modifiers to the spaceship
Clean OOP architecture Designed for maintainability, graded specifically on architecture quality
UI Systems Custom UI systems, library only included raw image rendering
Audio Systems Custom audio systems for sound effects and background music, library only included basic one-shot playback, which was not sufficient
