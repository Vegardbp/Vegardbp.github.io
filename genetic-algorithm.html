GPU Genetic Algorithm

Unity · C# · HLSL · Git

GPU-accelerated genetic algorithm running on the compute shader pipeline.

Solves the Rastrigin function across high-dimensional search spaces (500 dimensions with 10K individuals in 13 seconds on an RTX 3060).

Designed to be adaptable to any problem below 500 dimensions, tested with 3 dimensional PID-tuning with specified desired responses, solving in a few milliseconds.

Source code and report on GitHub

Demonstration Video

Details

Compute shader acceleration Entire fitness evaluation and selection pipeline runs on GPU via HLSL compute shaders
High-dimensional optimisation Solves the Rastrigin function — a standard benchmark for high-dimensional, multi-modal problems
Adaptable design Architecture is general enough to apply to other optimisation targets, such as PID tuning
Unity + HLSL Integrates Unity's compute shader pipeline with a C# driver and HLSL kernel
Custom tournament structure Custom method for fitness evaluation, allowing for parallelization of tournement selection, with CPU based final selection of absolute elites
GUI with parameter controls User interface for control of parameters, such as fitness thresholds, mutation rates, and more, with real-time visualisation of convergence and fitness distributions
